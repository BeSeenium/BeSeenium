String beseenium.action.elementActions.Click.execute ( int  n) throws ActionDataException, ClickException

This performs the click action. After successfully completing the action it also sets the List of WebElements in the ActionData object to null, to avoid attempting to click the same link twice in a row, as the second time the page would have changed and there may not be an identical element to click on.
