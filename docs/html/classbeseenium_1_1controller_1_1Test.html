Detailed Description

blueprint for creating Test objects, a Test is ostensibly a series of actions that are performed one after the other. provides methods for adding/removing actions configuring remote capabilities and other functionality that relates to creating and using a test.

this class acts as the client of command pattern governing the storage and execution of Abstract Actions.
