Detailed Description

A tag query is a child class of a generic Query, it represents a query that acts on a specific web element, this query can be chained with a terminating query.

String core.query.ChainableQuery.execute ( SelectionType  type,
String  searchParam,
String  returnParam 
) throws InvalidQueryException

executes the query defined by the parameters, the query may find more than one matching tag, this will be reflected in the returned string.

Parameters
typea type of selection appropriate defined in SelectionType enum: byId, byClass, byCss, byTagName, byName, byXpath, byLinkTxt, byPartialLinkTxt
searchParama string relevant to the selection method e.g. if selection method is byClass searchParam would be something like ".myClass"
returnParamthe attribute that you with returned i.e. "href", "class" and so on
Returns
String the value of the attribute defined in returnParam, if more than one matching element is found, the results of each element will be separated by a comma.
