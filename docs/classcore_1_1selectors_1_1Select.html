Detailed Description

This is a helper class that encapsulates some of the nasty finctionality away from the main structure. It provides methods that expose the Selection criterion for WebElements as well as the selection of terminating queries.

This provides a method of selecting particular WebElements given the selection criterion specified, this is a convenience that just encapsulates some inelegant code and strips it out of the ChainableQuery class.
